# Feynman AI Server

Backend service for the Feynman AI learning-by-teaching platform. It handles
real-time audio sessions over WebSockets, speech-to-text, retrieval, and
evaluation prompts that generate probing questions.

## Quick start

```bash
npm install
npm run dev
```

## Common scripts

- `npm run dev`
- `npm run worker:dev`
- `npm run dev:all`
- `npm run build`
- `npm run start`
- `npm run test`
